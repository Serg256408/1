Контент должен быть в HTML, а не генерироваться через JS (для SEO).

JS делает только поведение: аккордеон, подсчёт, скролл, подсветка меню.

Стили изолировать: обёртка #имя-чанка + префикс классов (rcf-, ps-).

Без onclick="" в HTML — только addEventListener в JS.

По возможности: стили в отдельный файл, JS в отдельный файл. Если нельзя — тогда внутри чанка, но строго: сначала <style>, потом HTML, потом <script>.

Адаптив обязателен: desktop + mobile (до 768px).

Никаких внешних библиотек без необходимости.# 1
