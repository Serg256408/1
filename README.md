# Калькулятор асфальтирования

Веб-калькулятор для расчёта стоимости работ по асфальтированию.

## Использование

Откройте файл `index.html` в браузере для использования калькулятора.

## Возможности

- Расчёт стоимости материалов и работ
- Три режима нагрузки: пешеходное, легковой транспорт, грузовая нагрузка
- Настройка всех параметров: песок, щебень, геоматериалы, асфальт, бордюры
- Автоматический расчёт земляных работ
- Адаптивный дизайн для мобильных устройств
