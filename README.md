Внутри чанка сначала идёт <style>...</style>, потом HTML-разметка, потом (если нужно) <script>...</script>.

Стили изолировать через корневой id блока (например #road-calc-seo), все селекторы только внутри него.

Никаких Tailwind-классов. Только обычный CSS + классы вида rcf-... / ps-....

Весь SEO-контент (заголовки, таблицы, тексты, цены, наименования) — в HTML, не генерировать через JS.

JS только для поведения: аккордеон, “показать ещё”, подсветка активной категории, пересчёт сумм.

Адаптив обязателен: desktop (сайдбар/таблица), mobile (удобная сетка/упрощённый вид).

Не использовать внешние библиотеки. Всё — в одном чанке, без зависимостей.

Код должен быть “копипастабельным” в MODX: без лишних <html>, <head>, <body>.
