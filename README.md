# Асфальтирование в Москве и МО

Веб-сайт для услуг асфальтирования в Москве и Московской области.

## Технологии

- MODX Revolution CMS
- PHP
- MySQL
- HTML/CSS/JavaScript

## О проекте

Сайт предоставляет информацию об услугах асфальтирования:
- Асфальтирование дорог
- Ремонт асфальтового покрытия
- Укладка асфальта
- Асфальтирование парковок
- Асфальтирование дворов

## Разработка

Проект использует MODX Revolution в качестве системы управления контентом. Подробные инструкции для GitHub Copilot доступны в `.github/copilot-instructions.md`.
