# Описание дизайна лендинга

## Общая концепция

Современный лендинг для услуги асфальтирования парковок, следующий дизайн-паттернам сайта lyapota.pro:
- Большие отступы между блоками
- Воздушная вёрстка с затемнённым фоном на баннере
- Жёлтые акценты (#FFD700) на кнопках и важных элементах
- Крупная типографика
- Белые карточки с тенями
- Анимации при наведении

## Цветовая палитра

- **Основной цвет (жёлтый)**: `#FFD700` - используется для кнопок, акцентов, бейджей
- **Тёмный фон**: `#1a1a1a` - для блоков 4, 10, 13
- **Вторичный цвет**: `#2c3e50` - для второстепенных кнопок
- **Светлый фон**: `#f8f9fa` - для чередования секций
- **Белый фон**: `#ffffff` - для карточек

## Блоки страницы

### 1. Hero Banner (Главный баннер)
- Высота: 600px
- Затемнённый фон (overlay 50% opacity)
- Большое фото: каток/парковка/техника
- Заголовок услуги крупным шрифтом (3rem)
- Подзаголовок с информацией о компании
- 2 кнопки: жёлтая "Рассчитать стоимость" и тёмная "Портфолио"

### 2. Мини-калькулятор
- Белая карточка с тенью на светлом фоне
- 4 шага с плавной анимацией переключения
- Иконки для каждого поля
- Жёлтая кнопка "Получить расчёт"
- Округления 12px

### 3. Преимущества (6 карточек)
- Сетка 3 колонки на десктопе
- Светлые карточки (#f8f9fa)
- Эмодзи иконки
- Анимация при наведении (поднимается на 5px)
- Тень увеличивается при hover

### 4. Описание услуги
- Тёмный фон (#1a1a1a)
- 2 колонки: текст слева, фото справа
- Белый текст
- Список с маркерами
- Закруглённое фото с тенью

### 5. Пироги покрытия (3 варианта)
- 3 карточки в ряд
- Средняя помечена как "Популярный" (жёлтый бейдж)
- Схема слоёв с номерами
- Жёлтые акценты на номерах слоёв
- Цена крупным шрифтом

### 6. Стоимость работ (кейсы)
- 3 карточки проектов
- Фото сверху (250px высота)
- Детальные параметры в таблице
- Итоговая стоимость жёлтым цветом
- Цена за м² мелким шрифтом

### 7. Этапы работы
- 6 карточек в ряд (горизонтальная лента)
- Номер шага в круге сверху (жёлтый)
- Эмодзи иконки
- Большие отступы между карточками
- Белые карточки на светлом фоне

### 8. Портфолио
- Мозаика 3×2 (6 проектов)
- При наведении: затемнение + текст появляется
- Увеличение фото при hover (scale 1.1)
- Информация об объекте в overlay

### 9. Отзывы
- 3 карточки
- Аватар с инициалами (жёлтый круг)
- Текст отзыва курсивом
- Фото объекта внизу карточки
- Белые карточки с тенью

### 10. Почему выбирают Транском
- Тёмный блок
- 4 больших цифры (жёлтый цвет, 4rem)
- 2 дополнительных блока с иконками
- Полупрозрачные карточки на тёмном фоне

### 11. Полный квиз (6 шагов)
- Белая широкая карточка
- Прогресс-бар (жёлтый)
- Большие кликабельные опции
- Эмодзи иконки для визуализации
- Плавная анимация переключения шагов

### 12. Форма заявки
- Белая карточка на светлом фоне
- Крупные поля ввода
- Жёлтая широкая кнопка
- Текст о конфиденциальности

### 13. Контакты + карта
- Тёмный футер
- 2 колонки: контакты слева, карта справа
- Яндекс карта (400px высота)
- Телефоны, email, адрес с иконками

## Адаптивность

- **Десктоп (>768px)**: полные макеты с колонками
- **Планшет (768px)**: 2 колонки или 1 колонка
- **Мобильный (<480px)**: все блоки в 1 колонку

## Типографика

- **Заголовки H1**: 3rem (48px), font-weight: 800
- **Заголовки H2**: 2.5rem (40px), font-weight: 700
- **Заголовки H3**: 1.5rem (24px), font-weight: 700
- **Текст**: 1rem (16px), line-height: 1.6
- **Шрифт**: system-ui (Apple/Google native fonts)

## Анимации

- Плавные переходы: `transition: all 0.3s ease`
- Поднятие при hover: `transform: translateY(-5px)`
- Увеличение фото: `transform: scale(1.1)`
- Появление контента: `animation: fadeIn 0.5s ease`

## Особенности

- Все формы интегрированы с MODX FormIt/AjaxForm
- UTM-метки автоматически отслеживаются
- Маска телефона: +7 (___) ___-__-__
- Яндекс.Карты с меткой компании
- Smooth scroll для якорных ссылок
