# MODX Chunks Integration Guide

Этот проект содержит 13 модульных блоков для лендинга услуги асфальтирования парковок.

## Структура файлов

- `landing-page.html` - Полная HTML страница со всеми блоками
- `styles.css` - CSS стили для всех блоков
- `script.js` - JavaScript для интерактивных элементов
- `modx-chunks/` - Директория с отдельными чанками для MODX

## Как использовать в MODX

### Вариант 1: Полная интеграция

1. Создайте новый ресурс в MODX
2. Скопируйте содержимое `landing-page.html` в контент ресурса
3. Добавьте CSS в `<head>` секцию шаблона или создайте отдельный файл стилей
4. Добавьте JavaScript перед закрывающим `</body>` тегом

### Вариант 2: Модульная интеграция

1. Создайте отдельные чанки в MODX для каждого блока
2. Вызывайте чанки на странице через `[[$chunkName]]`
3. Создайте общий файл стилей и подключите его в шаблон

## Список блоков (чанков)

1. **Hero Banner** - Главный баннер с кнопками
2. **Calculator** - Мини-калькулятор расчёта стоимости (4 шага)
3. **Advantages** - 6 карточек преимуществ компании
4. **Service Description** - Тёмный блок с описанием услуги
5. **Pavement Layers** - 3 варианта пирогов покрытия
6. **Pricing** - 3 примера реализованных проектов со стоимостью
7. **Stages** - 6 этапов работы (таймлайн)
8. **Portfolio** - Галерея из 6 проектов с наведением
9. **Reviews** - 3 отзыва клиентов с фото
10. **Why Us** - Блок "Почему выбирают Транском" (тёмный фон)
11. **Quiz** - Полный квиз из 6 шагов с прогресс-баром
12. **Contact Form** - Форма заявки
13. **Footer** - Контакты и карта

## Интеграция с FormIt/AjaxForm

Для форм используйте следующий синтаксис вызова:

```modx
[[!AjaxForm?
    &snippet=`FormIt`
    &form=`calculatorForm`
    &hooks=`email`
    &emailTpl=`emailCalculator`
    &emailSubject=`Заявка с калькулятора`
    &emailTo=`your@email.com`
    &validate=`name:required,phone:required`
]]
```

## Настройка

### Цвета
Основной цвет (жёлтый): `#FFD700`
Темный фон: `#1a1a1a`
Вторичный цвет: `#2c3e50`

### Изображения
Замените пути к изображениям на реальные:
- `images/hero-bg.jpg` - Фон баннера
- `images/parking-asphalt.jpg` - Фото для описания
- `images/project-*.jpg` - Фото проектов
- `images/portfolio-*.jpg` - Фото портфолио
- `images/review-*.jpg` - Фото для отзывов

### Яндекс Карты
Добавьте свой API ключ в script.js:
```html
<script src="https://api-maps.yandex.ru/2.1/?apikey=YOUR_API_KEY&lang=ru_RU"></script>
```

## Особенности дизайна

- Большие отступы между блоками (80px padding)
- Закругления углов 12px
- Анимации при наведении (translateY, scale)
- Затемнённый фон на баннере
- Жёлтые акценты на кнопках и важных элементах
- Адаптивный дизайн для мобильных устройств

## Требования

- MODX Revolution 2.x или выше
- Дополнения: FormIt, AjaxForm (опционально)
- Современный браузер с поддержкой CSS Grid и Flexbox

## Поддержка

Все блоки полностью модульные и могут быть использованы независимо друг от друга.
Каждый блок можно легко адаптировать для других услуг, изменив только текстовый контент.
